# Автоматизация развертывания Active Directory

- Создание нового домена в новом лесу
- Добавление нового домена дерева в существующий лес
- Добавление дополнительного контроллера в существующий домен
- Настройка DNS и сайтов

На каждом сервере необходимо настроить сеть и запустить `srv_preinstall.ps1` для настройки RDP и изменения имени.

### New_Forest.ps1
Создание нового домена в новом лесу.

>`$Domain_Name` - имя домена  
>`$DSRM_password` - DSRM пароль 

### Root_DC_DNS_Sites.ps1
Добавление DNS Conditional Forwarder для домена дерева и создание сайтов после создания корневого домена.

>`$name` - имя домена дерева  
>`$IP` - IP-адрес домена дерева  
>`$subnet1` - подсеть  
>`$SiteName1` - название первого сайта  
>*Аналогично можно добавить другие сайты и подсети.*

### New_Tree_Domain.ps1
Создание нового домена дерева в существующем лесу.

>`$Domain_Name` - имя домена дерева  
>`$Root_Domain` - имя корневого домена  
>`$DSRM_password` - DSRM пароль   
>`$SiteName` - название сайта для нового DC (сайт уже должен существовать)  

### Tree_DC_DNS.ps1
Добавление DNS Conditional Forwarder для корневого домена после создания нового домена.

>`$name` - имя корневого домена  
>`$IP` - IP-адрес корневого домена  

### Add_DC_to_Domain.ps1
Добавление дополнительного контроллера домена в существующий лес.

>`$Root_Domain` - имя домена  
>`$DSRM_password` - DSRM пароль  
>`$SiteName` - название сайта для нового DC (сайт уже должен существовать)
